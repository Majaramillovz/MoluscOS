PROYECTO: RUANA DE MYLAR

Ante el "Super-Niño" y la vigilancia corporativa, la lana es obsoleta. Necesitamos reflejar el 99% de la radiación solar y disolver la firma de calor ante sensores infrarrojos.

[!] ADVERTENCIA: RUIDO AL MOVERSE. NO APTO PARA ZONAS DE SILENCIO. RIESGO DE GOLPE DE CALOR SIN VENTILACIÓN.

MATERIALES Y CONCEPTO

Se requieren 2 mantas térmicas de emergencia (plateadas), cinta de ducto negra (la transparente se derrite) y 4 monedas para lastre.

. . /' \ /`\ / | .---. | \ | |/ _ \| | | |\ _ /| | | | '---' | | | | | | / | | \ | | | | |______| |______| MYLAR_V3 // ANTI-DRONE
  1. Fusión: Extiende las mantas. Une el lado largo con cinta de ducto por ambas caras para crear un cuadrado de 2x2m.
  2. Cuello: En el centro exacto, corta una hendidura de 30cm. Refuerza los bordes inmediatamente o se rasgará.
  3. Lastre: Pega monedas en las 4 esquinas. Sin esto, el Mylar vuela como papel y no cubre el cuerpo.
  4. Ventilación: Haz cortes en "V" bajo las axilas. Vital para evitar cocinarte en tu propio sudor.
