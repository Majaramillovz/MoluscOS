Antena Dipolo Multibanda

[TIPO: NIVEL 1 - TÉCNICA REAL]
[ESTADO: FUNCIONAL / PRE-COLAPSO]
[FUENTE: QST / ARRL / DIY ARCHIVES]

Este diseño utiliza cintas métricas metálicas como elementos radiantes. Su ventaja táctica es la capacidad de ajustar la frecuencia (sintonía) simplemente retrayendo o extendiendo la cinta, sin necesidad de cortar cables.

01 // MATERIALES (SCAVENGER LIST)
  • 2 x Cintas métricas (Flexómetros) de 5m o más.
  • 1 x Bloque de madera o PVC (Aislante central).
  • 1 x Núcleo de ferrita toroidal (Recuperado de fuentes de poder viejas).
  • 1 x Cable coaxial RG-58 (Bajada a la estación).
  • 1 x Caja de empalme impermeable.
  • Cuerda de driza y polea simple.
FIG 1.0: ESQUEMA DE MONTAJE Y BALUN [ARCHIVO RECUPERADO: 5516t9]
02 // EL BALUN 1:1 (CHOQUE)

Es vital para evitar que la línea de transmisión (el cable coaxial) irradie señal y cause interferencia.

Instrucción: Enrolla el cable coaxial (o alambres esmaltados conectados al conector SO-239) alrededor del toroide de ferrita. 8 a 10 vueltas son suficientes para HF.

03 // CÁLCULO DE RESONANCIA

Para transmitir, la antena debe medir media longitud de onda de la frecuencia deseada. Usa esta fórmula para saber cuánto extender cada cinta métrica:

L (metros por lado) = 71.25 / Frecuencia (MHz)

Ejemplo Práctico (Banda de 40m / 7.1 MHz):
71.25 / 7.1 = 10.03 metros por cada lado.

04 // INSTALACIÓN DE GUERRILLA

Utiliza el esquema de polea en árbol (ver Fig 1.0 derecha). Esto permite bajar la antena rápidamente para mantenimiento, ajuste de frecuencia o para ocultarla ante patrullas de detección.

¡PELIGRO DE ALTA TENSIÓN!
Las puntas de las cintas acumulan alto voltaje RF al transmitir.
Mantener alejado del contacto humano durante la operación.
