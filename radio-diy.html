MANUAL: ANTENAS DE EMERGENCIA

Si no hay satélites, capturamos el aire. Aquí tienes dos diseños de "alta ganancia" fabricados con basura doméstica. Funcionan para TV Digital (UHF) y Radio FM.

OPCIÓN A: LA PAJARITA (BOWTIE)

La reina de las antenas DIY. Materiales: Perchas de ropa (metal), tabla de madera, tornillos.

\ / \ / <- 20cm -> \ / \ / \ / \ / O-----X-----O <- Tornillos / \ / \ / \ / \ / \ / \
  1. Corte: Corta 2 trozos de alambre de 40cm cada uno.
  2. Doblado: Dobla cada alambre en forma de "V" o ">". La apertura de las puntas debe ser de unos 20cm.
  3. Montaje: Atornilla los vértices de las "V" a una madera, separados por 3-5 cm.
  4. Conexión: Pela el cable coaxial. El núcleo de cobre va a un tornillo (lado izquierdo) y la malla trenzada al otro (lado derecho).

OPCIÓN B: DIPOLO DE LATA

Ancho de banda brutal. Materiales: 2 Latas de cerveza/refresco, cinta aislante, palo o percha.

[=======] [=======] | LATA | | LATA | | A | | B | [=======] [=======] | | | | (Núcleo) (Malla) \ / \ / COAXIAL
  1. Preparación: Lija la parte donde está el agujero de beber para quitar el barniz protector (el aluminio debe brillar).
  2. Montaje: Pega las latas en un palo con cinta, con los agujeros enfrentados pero sin tocarse (deja 2cm de aire).
  3. Conexión: Enrolla el cable de cobre del coaxial en la anilla de la Lata A. Enrolla la malla del coaxial en la anilla de la Lata B.
  4. Aislamiento: Cubre todo con cinta para evitar corrosión.
